# PPC Optimizer

Goal

Build a production-ready application that analyzes Google Ads Product reports.

Tech stack

- Python 3.12+
- pandas
- openpyxl
- pydantic
- typer
- loguru
- pytest

Rules

- Modular architecture.
- SOLID.
- Type hints everywhere.
- One responsibility per module.
- No duplicated code.
- No hardcoded paths.
- English comments and docstrings.
- Ukrainian UI labels are allowed only in Excel reports.

Output

The application must generate report.xlsx.
